# Assets

`icon.png` is the canonical Lattice product icon master. Product surfaces should
reference this file where possible instead of keeping separate icon copies.

`icon.icns` is generated from `icon.png` for the macOS app bundle.

`github-social-preview.jpg` is the 1280 x 640 social preview image for the
GitHub repository.

`png/` contains bitmap exports for surfaces that cannot use SVG:

- `icon.png`: 512 px
- `icon-256.png`
- `icon-128.png`
- `icon-64.png`
- `icon-32.png`
- `icon-16.png`
