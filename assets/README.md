# Assets

`icon.svg` is the canonical Lattice product icon. Product surfaces should reference this file where possible instead of keeping separate icon copies.

`png/` contains bitmap exports for surfaces that cannot use SVG:

- `icon.png`: 512 px
- `icon-256.png`
- `icon-128.png`
- `icon-64.png`
- `icon-32.png`
- `icon-16.png`
